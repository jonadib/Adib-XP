# Adib XP

A Windows XP-themed portfolio website built with React, Tailwind CSS, and Vite.

## Features

- **Windows XP Aesthetic**: Authentic-looking desktop, taskbar, start menu, and windows.
- **Projects Explorer**: Browse projects with a custom IE6-style navigation bar and preview screenshots.
- **Skills Blueprint**: A tactical skills section with a grid background.
- **About Me**: A detailed bio and social links section styled like the XP Explorer pane.
- **Dark Mode**: Supports light and dark themes (restricted to the Projects section).

## Tech Stack

- **Framework**: React 18
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Build Tool**: Vite

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Build for production: `npm run build`
